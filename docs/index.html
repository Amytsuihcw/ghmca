成立宗旨

以音為引，服務社會

全球健康音樂慈善總會於2024年由黃騫緯博士及鍾麗裳博士共同創立，秉承「以音樂為引，服務社會為本」的理念，致力於：

  • 推廣音樂治療，提升公眾對音樂療癒力量的認識
  • 為長者、病患及弱勢社群提供免費音樂服務
  • 培育音樂治療專業人才，推動行業發展
  • 聯繫全球音樂慈善力量，共建關愛社區
